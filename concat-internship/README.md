# concat-internship
2022.12 - 2023.2 콘캣 주식회사 인턴십 (인공지능 개발자)

## 프로젝트
약 38,000 건의 헬스케어 상담 녹취록 분석 및 insight 도출

### Whisper
- open AI가 만든 STT 오픈 소스 라이브러리 
- 헬스케어 상담 녹취록을 구문 단위로 분리된 텍스트로 변환하기 위해 사용

### Bertopic
- topic modeling 수행이 가능한 모델
- BERT 기반 임베딩 + class based tf-idf 이 핵심
- 헬스케어 상담 녹취록의 주요 토픽을 뽑아내기 위해 사용
